# Health Desk Clinic Website

A modern, responsive health supplements e-commerce website built with React and Tailwind CSS.

## Live Demo

Check out the live site here: [Health Desk Clinic](https://tarunthedev.github.io/healthdesk-figma-to-code/)

## About This Project

This project is a Figma-to-code conversion of a health supplements website. It showcases a clean, professional design for a fictional health products company called "Health Desk Clinic".

### What's Included

- Hero section with call-to-action
- Product categories with horizontal scroll
- Best sellers showcase
- Customer testimonials
- Trust badges and certifications
- Newsletter signup
- Responsive footer with contact info

## Tech Stack

- **React** - For building the user interface
- **TypeScript** - For type-safe code
- **Tailwind CSS** - For styling
- **Vite** - For fast development and building
- **Shadcn UI** - For pre-built UI components

## Getting Started

### Prerequisites

Make sure you have Node.js installed on your computer. You can download it from [nodejs.org](https://nodejs.org/).

### Installation

1. Clone this repository:
```bash
git clone https://github.com/TarunTheDev/healthdesk-figma-to-code.git
```

2. Navigate to the project folder:
```bash
cd healthdesk-figma-to-code
```

3. Install the dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and go to `http://localhost:5173`

## Building for Production

To create a production build, run:

```bash
npm run build
```

The built files will be in the `dist` folder.
